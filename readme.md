# Wisecore Orchestrator

## Requisitos
- Docker
- Docker Compose

## Instalación
1. Clonar este repositorio
2. Ajustar el archivo de variables de entorno
3. Ejecutar:
   ```bash
   docker compose up -d
4. Acceder:
    - Backend: http://localhost:8000/docs
    - Frontend: http://localhost:3000
